# 위치 필터 기능 기획서

## 개요

사용자가 원하는 지역의 판매자 상품만 볼 수 있도록 위치 필터를 제공합니다.
지역은 **2단계(시/도 → 시/군/구)** 로 선택하며, 인구가 적은 지역은 묶음으로 표시합니다.

> 🔵 개별 항목 / 🟡 묶음 항목

---

## 서버 전달 규칙

| 선택 조합 | depth1 | depth2 | 의미 |
|-----------|--------|--------|------|
| 전국 > 전국 전지역 | `null` | `[]` | 전국 필터 없음 |
| 전국 > 서울 | `"서울특별시"` | `[]` | 서울 전체 |
| 서울 > 전지역 | `"서울특별시"` | `[]` | 서울 전체 |
| 서울 > 강남구 | `"서울특별시"` | `["강남구"]` | 강남구만 |
| 경기 > 전지역 | `"경기도"` | `[]` | 경기 전체 |
| 경기 > 수원시 | `"경기도"` | `["수원시"]` | 수원시만 |
| 경기 > 가평·양평 | `"경기도"` | `["가평군", "양평군"]` | 가평 + 양평 |
| 세종 > 전지역 | `"세종특별자치시"` | `[]` | 세종 전체 |
